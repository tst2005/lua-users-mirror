lua-users.org is an internet site for and by users of the programming language Lua.

To find out what Lua is and for documentation, downloads, mailing list info, etc. please visit the official Lua web site. The lua-users.org web is not affiliated with Tecgraf.

Currently available resources:

  • lua-users wiki - collaborative web site
  • lua-l archive - searchable web archive of the official Lua mailing list
  • lua-users chat room - IRC channel (#lua at irc.freenode.net)
  • lua-users.org mini charter
