Lua Profiler
wiki

LuaProfiler is a time profiler for Lua.

Description: This can visually present you to the calls and functions that make your program slow.

Web site: http://luaforge.net/projects/luaprofiler . There is an older version [1].

Documentation: See [LuaProfiler documentation]. There also some LuaProfilerExamples for the older version.
