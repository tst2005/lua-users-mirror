Lua Curl
wiki

There have been three bindings to [cURL libcurl], which allows getting or sending files from URLs.

  • luacurl by AlexanderMarinov. [luacurl.luaforge.net]; [git clone of tarballs (on LuaDist)]; [git clone of tarballs + LuaDist build (on LuaDist)]
  • freepops-luacurl by Enrico Tassi - This is used in [FreePOPs]. [git clone of tarballs (on LuaDist, "freepops" branch)].
  • Lua-cURL by Jürgen Hötzel. [lua-curl.luaforge.net]; [git]
  • Note: As noted in [1], luacurl (and freepops-luacurl) adopts the easy interface, while LuaCURL is aiming for a full-fledged libcurl binding (easy/multi/share interface). luacurl and freepops-luacurl share a luaforge project page and are also in the same LuaDist "luacurl" git repo (in separate branches), but they appear to have been independently developed. luacurl exists as a package in LuaRocks, LuaDist, and LuaForWindows.
