Meta Programming
wiki

Here's various links relating to metaprogramming[1][2] in Lua. This includes macros, token filters, facilities for syntax extensions, and source processors.

  • MetaLua - An alternative Lua compiler supporting compile-time metaprogramming and syntax extensions. It also contains a pure Lua library for parsing Lua source into ASTs (FabienFleutot). [comparison to luaSub and Luma]
  • [Luma] - LPeg-based Lua Macros, inspired by Scheme's define-syntax/syntax-rule (FabioMascarenhas)
  • LuaMacros - macro facility for Lua using token filters (SteveDonovan)
  • [tokenf patch] - adds token filters to Lua (LuizHenriqueDeFigueiredo) [presentation]
  • [ltokens] - A library for lexical analysis of Lua code based directly on the Lua lexer (LuizHenriqueDeFigueiredo)
  • [luaSuper] - syntax modifiable Lua front-end (AskoKauppi) (luaSub)
  • LuMaX[3] - loosely based on LuaMacros (ThomasLauer)
  • LuaFish - Parse Lua to AST via LPeg, with experimental macro support (DavidManura), also LuaAnalyze - uses MetaLua libraries
  • SourceOptimizer - optimizer that inlines Lua functions, using MetaLua libraries (DavidManura)
  • FileLineMacros - simple C token filter implementing C-like __FILE__ and __LINE__ macros
  • SciteMacroExpander - expands macros in the SciTE text editor (SteveDonovan)
  • SourcePreprocessing - some notes on source filters
  • CodeGeneration - runtime code generation
  • FeatureProposals - a built-in macro facility has many times been suggested for Lua
  • LuaGrammar - facilities for lexing/parsing Lua
  • search the [lua-l mail list] for many discussions on metaprogramming, macros, token filters, etc.

See also

  • CppMetaprogramming - C++ metaprogramming using Lua (design pattern, module)
