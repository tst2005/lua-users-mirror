Lua Development Model
wiki

This page outlines topics on the Lua development model. It addresses questions like

  • Where is the Lua source code repository?
  • Where is the test suite?
  • Why don't you add X to the language?
  • How does the language evolve?

Official Lua source:

  • http://www.lua.org/download.html + any patches at http://www.lua.org/bugs.html

Non-official source control repositories:

  • [repo.or.cz/w/lua.git] - git repository of Lua versions >= 1.0 [discussion]
  • [lua5_history] - Mercurial repository of Lua versions >= 5.0

Lua test suite:
