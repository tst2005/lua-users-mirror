World Of Warcraft
wiki

Lua is used in the game World of Warcraft to allow players to create their own custom addons (aka mods) that enhance, add to, or otherwise modify the default game interface.

You can find many of these addons at the site WoWInterface.com:

  • http://www.wowinterface.com/

The actual base framework for World of Warcraft's interface is written in Lua, and can be extracted from the game's data files using a tool provided by Blizzard.

More information about WoW's use of Lua and the interface API can be found on WoWWiki.com:

  • http://www.wowwiki.com/Lua
  • http://www.wowwiki.com/World_of_Warcraft_API

There is a distribution of Lua available (created by cogwheel) that includes everything required to install Lua locally for addon development:

  • http://sourceforge.net/projects/lua511wow/

This includes wowlib, luasocket, and the standard Lua 5.1.1 Win32 binaries.

There has also been a book written about writing addons for World of Warcraft, called "Hacking World of Warcraft" by Daniel Gilbert and James Whitehead. It is available through [Amazon.com].

Also see #wowace and #wowi-lounge on irc.freenode.net. Archives are available at the following URLs:

  • http://purl.rikers.org/%23wowace/
  • http://purl.rikers.org/%23wowi-lounge/
