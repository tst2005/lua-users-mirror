Lua Language
wiki

This directory contains discussions about the Lua programming language, its implementations, and modifications.

  • Language Definition
    • LuaShortReference - concise summary of the Lua 5.1 syntax and core libraries
    • LuaReferenceManualComments - user comments and annotations on the Lua Reference Manual
    • Evolving
      • FeatureProposals - discussions about new features or feature changes
      • LuaFiveTwo - notes on the next major version of Lua (5.2)
      • MigratingToFiveOne - tips for users moving from 5.0 to 5.1
      • LuaFiveFeatures - summary of changes between Lua 4.0 and Lua 5.0
  • Features and Know-How
    • TutorialDirectory - various tutorials written and reviewed by Lua users
    • GarbageCollection - info on Lua's garbage collection
    • LuaScoping - on variable scope and upvalues
    • VarargTheSecondClassCitizen - on using a vararg expression
    • HiddenFeatures - describes unsupported and undocumented features of the standard Lua distribution
    • FloatingPoint - explains why double floating point is perfectly fine for most integer applications
    • CountingFromOne - the use of ordinals instead of offsets, pros and cons
      • IndexIntervalFormatIssue - more on 1-based indexing and closed vs. right-open interval slicing
    • LuaVirtualization - which operations can/cannot be hooked
    • The Lighter Side of Lua
      • LuaTrivia - test your Lua knowledge
      • ZenOfLua - grok it
  • Enhancement and Modification
    • General Help
      • CompilerWritingReferences - info on compilers, may be useful if someone would like to edit the Lua source code
      • LuaSource - comments on the source code and implementation of Lua
      • LuaSourceTable - some notes about Lua's table implementation
    • Approaches
      • LuaPowerPatches - "mix and match"-friendly patches to the official Lua distribution
      • ModifyingLua - example of changing Lua itself
      • MetaLua - alternative Lua compiler with compile-time metaprogramming and syntax extensions
      • InteractiveLua - a friendlier interactive interpreter
      • LuaShell - an incomplete sample shell which understands Lua syntax
      • CompleteWithReadline - completion engine for standalone interpreter
      • LuaCompilerInLua - reimplementing the Lua compiler front-end (luac.c) in Lua (program)
      • LuaInterpreterInLua - reimplementing the Lua interpreter (lua.c) in Lua (program)
  • Metaprogramming
    • See LuaDomains.
