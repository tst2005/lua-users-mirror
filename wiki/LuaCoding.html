Lua Coding
wiki

This pages discuss several kinds of source code related topics. This includes code editing, building, binding, testing, optimizing, maintaining, and sharing.

  • LuaDevelopmentModel - test suite, source code repositories, and discussions

  • Editing and Formatting Lua Code
    • LuaIntegratedDevelopmentEnvironments - IDEs and editors
    • LuaEditorSupport - syntax highlighting, auto-formatting, etc.
    • SciteScripts - Lua scripts for the SciTE text editor
    • SourceCodeFormatter - may be helpful to format your code for the wiki
    • VmMerge - merges Lua source code with VM disassembly
  • Source Preprocessing / Metaprogramming
    • See LuaDomains.
  • Building and Binding
    • BuildingLua - notes on compiling Lua and some build packages
    • BuildingModules - how to properly build C extension modules for Lua
    • BindingCodeToLua - how to call or bind C or C++ code to Lua
    • LuaBuildSystems - build systems implemented or expressed in Lua
    • PlatformSpecific - platform-specific breakdown of the Lua universe
    • LuaProxyDll - create a proxy DLL for statically linked Lua
    • EasyManualLibraryLoad - greatly simplifies manual loading of Lua 5.1 shared library
    • AutoConfig - autoconfig macros for lua 5.0 and 5.1
    • BinToCee - generate a C file to load and run a Lua program
  • Modules and Packages
    • LuaForge - contains modules (and applications)
    • LuaRocks - installs Lua modules as self-contained packages (with dependency info) called "rocks"
    • LuaSearch - an effort to provide a centralized locater of Lua modules and documentation (like CPAN)
    • ModuleReview - an effort for classification, quality assurance, and standartization of existing Lua modules
    • SampleCode - modules and code snippets on this wiki
    • LibrariesAndBindings - useful modules from elsewhere
    • Module Programming
      • See LuaProgramming.
  • Quality and Testing
    • UnitTesting - how to write unit tests with Lua
    • DebuggingLuaCode - tools for Lua code debugging
    • LuaSecurity - security issues in scripts
    • DetectingUndefinedVariables - detecting undefined globals
    • LuaTypeChecking - implementing type checking in Lua (design patterns and code)
    • LuaStyleGuide - recommended Lua coding practices
  • Performance and Optimization
    • OptimisationTips - making Lua code more efficient
    • ProfilingLuaCode - some profilers for Lua code
    • OptimizedSampleCode - best known algorithms that perform the specified tasks

See also

  • LuaAddons - a directory of "things for Lua" that are outside this site
    • LuaTools - source processors, builders, bytecode (dis)assemblers, etc.
  • LuaDomains - application-field specific approaches, techniques, and modules
  • LuaLinks - a directory of external Lua links including articles, tutorials, and comparisons
