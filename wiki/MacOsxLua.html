Mac Osx Lua
wiki

This is an index of the Lua universe on Mac OS X. Distributions, uses and tips are listed here.

For Lua on Mac OS Classic, see MacOsClassicLua.

Binaries and native builds

  • See [LuaForge Lua Binaries project] for the most recent binaries for Mac OS X.
  • [Rudix] has packages for 10.6, 10.7, and 10.8.
  • [Lua for Mac+Windows] (5.1) - OS X 10.3+ Intel/PowerPC builds.
  • [Lua Framework Build] (5.0.2) - binary and source available.
  • DaleeLua - Lua 4.0 for Mac OS X, distribution by David Lee. (original page gone; now hosted on this wiki)

Distributions

  • [Fink packages] - lua51 is in stable, do sudo fink install lua51. See also [Fink].
  • [MacPorts] - lua 5.2 is similarly available through MacPorts?. Do "sudo port install lua".
  • [Homebrew formula] - do "brew install lua".
  • See also multi-platform LuaDistributions.

Development Environments

  • ? LuaIntegratedDevelopmentEnvironments

Frameworks

  • [LuaObjCBridge] - open source framework for embedding a Lua interpreter into Objective-C (Cocoa) programs.
