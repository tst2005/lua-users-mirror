Lua Com
wiki

LuaCOM is a library allowing Lua to use and implement Microsoft's [Component Object Model (COM)] / ActiveX objects.

Dependencies: The current release version, 1.4, is compatible with Lua 5.1

Feedback: Inquiries can be directed to the luacom-developers mail list, as well as forums and bug tracker. The general Lua mail list [1] has some posting on LuaCOM too.
