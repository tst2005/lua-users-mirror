My name is Bertrand Mansion, I live in Paris, France. I tought myself Lua when writing a WoW addon... I find the language very efficient and elegant, especially compared to other scripting languages like PHP, Python, Ruby. It's lightweight, yet very powerful and extensible. As I mostly do web development, I am interested in mod_lua which will probably be bundled with Apache 2.4. I would like Lua to offer more libraries for web development. As such, I am going to contribute some open-source code to make web development with Lua easier.

In the meanwhile I have created a website where you can try Lua online directly, it's called [Try Lua!]
