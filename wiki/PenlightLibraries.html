Penlight Libraries
wiki

[Penlight] is a set of pure Lua libraries capturing common tasks like file system operations, configuration file reading, functional utilities (like map), application support, and a basic class model. It was originally inspired by the Python standard library, and only directly depends on LuaFileSystem.

The documentation is available [here].

It is available using LuaRocks and will be part of the upcoming Debian release.
