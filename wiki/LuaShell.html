Lua Shell
wiki

Lsh is a preliminary system shell which understands large part of Lua syntax.

Description: It is built on top of a Lua patch in which the parser tries to feed the runtime environment of Lua with some information over statements which it could not parse. A Lua initialization script interprets them.

As of now it features simple filename completion, dynamic prompt and aliases.

Installation notes are in the tarball INSTALL file. Please understand this is very incomplete and buggy software and is not meant for anything but development yet.
