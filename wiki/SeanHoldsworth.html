I'm a self confessed computer language junky who stalks the "Lambda the Ultimate" website whilst trying to stop my brain frying with some of the concepts discussed there.

I keep coming back to Lua as an interesting niche language that deserves to be more widely known. I've used a huge spread of computer languages from hand coding machine code on early microprocessors, through battling programmer visible pipelines on Intel i860s, the usual spread of C, C++, Java sequence, did research using functional languages such as Haskell, and have ended up these days writing most of my code in Python and Erlang.

My current work involves writing highly concurrent systems for network applications.
