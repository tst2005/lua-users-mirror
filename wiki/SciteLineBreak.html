Scite Line Break
wiki

SciTELineBreak is a Lua extension for the SciTE editor.

Description: This extension/add-on (started using SciteHexEdit as example) adds a command, that toggles "line breaking" while typing characters (but not backspaces or copy-paste). For full description and install details, see the script itself.

Download: The extension script can be downloaded from:

  • http://sdaaubckp.svn.sourceforge.net/viewvc/sdaaubckp/extensions/scite/SciTELineBreak.lua

(you can add [?content-type=text%2Fplain] and [?view=markup] to that URL to browse)
